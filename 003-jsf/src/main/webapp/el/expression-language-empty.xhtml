Expression Language : operateur "empty"

empty / not empty

Le mot clé "empty" permet de vérifier si une variable est vide ou null.

Sur une chaine ou tout autre objet

Par exemple, empty retournera true si une chaine vaut "" ...

... ou null

Sur une collection

L'opérateur empty a un comportement spécifique sur une collection ou un tableau. Si la collection ou le tableau est null ou vide (aucun élément dans la collection), il retournera true.
